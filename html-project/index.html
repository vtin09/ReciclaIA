Projeto TCC - Reciclagem Inteligente com Tecnologia (Reciclaí)

Este projeto propõe um aplicativo inovador para incentivar a reciclagem de lixo, contribuindo diretamente para a preservação ambiental. Essa iniciativa está alinhada com os Objetivos de Desenvolvimento Sustentável (ODS) da ONU, promovendo a conscientização ambiental e incentivando a redução do desperdício. Além disso, a reciclagem é uma estratégia essencial para reduzir a quantidade de lixo enviada para os aterros sanitários, minimizando a poluição e seus impactos ambientais.
